Event Planning

Given a set $H$ of hotels, denote the nightly rate for hotel $h$ as $R_h$. Denote the set of weekend vacancies for hotel $h$ by $V_h$.

The set $C$ of hotels with enough capacity for all $N$ participants on at least one weekend is defined as $$C := \{ h \in H \space | \space \exists n \in V_h [ n \ge N ] \}.$$ Let $\beta \subseteq C$ be the set of hotels whose nightly rate keeps us within our budget $B$: $$\beta := \{ h \in C \space | \space N \cdot R_h \le B \}$$ Then the cheapest cost $A$ for the trip ($-1$ if nothing is available) is $$A := \begin{cases} N \cdot \min_{h \in \beta} R_h, & \text{if $\beta \ne \emptyset$ } \\ -1, & \text{otherwise} \end{cases}$$
