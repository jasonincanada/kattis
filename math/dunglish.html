Dunglish

We have a set $W$ of words from which a sentence $S$ has been constructed. For example: $$\begin{align} W &:= \{ \text{als}, \text{mollen} \} \\ S &:= \text{'als mollen mollen mollen'} \end{align}$$ The words can occur more than once in the sentence, so define a function $s : W \to \mathbb{N}$ that maps a word to the number of occurrences of it in $S$. For the above example: $$\begin{align} s(\text{als}) &= 1 \\ s(\text{mollen}) &= 3 \end{align}$$

Define two maps: $c : W \to \mathbb{N}$, which maps words to the number of correct translations in our dictionary; and $t : W \to \mathbb{N}$, which maps words to the total translations (both correct/incorrect). In this section we will be computing totals only, not determining the translations themselves, so the English words are not represented here.

The total possible correct translations $C$ of the sentence $S$ is given by: $$C := \prod_{w \in W} c(w)^{s(w)} $$ The total possible translations $T$ is: $$T := \prod_{w \in W} t(w)^{s(w)} $$ The number of incorrect translations $I$ is then: $$I = T - C.$$
